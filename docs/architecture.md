# Архитектура GANYMED Blockchain

## Обзор системы

GANYMED (ГАНИМЕД) — блокчейн-платформа с гибридным консенсусом (PoA/PoS), смарт-контрактами и стандартом токенов GND-st1. Подключение к ноде: **main-node.gnd-net.com**; документация API: **api.gnd-net.com**. Полный перечень реализованных сервисов: **[services.md](services.md)**.

### Основные компоненты (реализовано)

1. **Core Node**
   - Консенсус (PoA/PoS), выбор по транзакции
   - Блокчейн (генезис, блоки, загрузка из PostgreSQL)
   - Состояние (State), мемпул, кошельки, токены, контракты
   - Метрики и алерты (core.Metrics)

2. **API Layer**
   - REST API (порт 8182, `/api/v1/*`)
   - WebSocket API (порт 8183)
   - RPC API (порт 8181)

3. **Storage**
   - PostgreSQL (основное хранилище)

4. **Monitoring и аудит**
   - Метрики (core + monitoring.MetricsRegistry)
   - Алерты (AlertManager), события (EventLogger)
   - Аудит: монитор подозрительных транзакций, правила, проверки безопасности контрактов
