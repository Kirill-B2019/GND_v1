# Безопасность

Рекомендации и политики безопасности блокчейна ГАНИМЕД.

## Общие положения

- Защита ключей и кошельков, использование подписей транзакций.
- Валидация контрактов и проверки в модуле аудита (см. [services.md](services.md)).
- Безопасность API: ограничение частоты запросов, валидация входных данных.
