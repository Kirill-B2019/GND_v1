# Модули GNDst-1

Каталог для контрактов-расширений (модулей) стандарта GNDst-1.

Модули регистрируются в токене через `registerModule(moduleId, moduleAddress, name)` (вызов от контроллера). Вызов логики модуля выполняется через `moduleCall(moduleId, data)`.

Сюда помещают Solidity-контракты, реализующие дополнительную функциональность (голосование, стейкинг, верификация и т.д.), адрес которых затем передаётся в `registerModule`.
